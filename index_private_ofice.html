    что такое
    открытый
    регион 71?

    Для достижения поставленных задач с привлечением экспертных сообществ правительством Тульской области разработана пошаговая «Дорожная карта» (на период с 2013 до 2016 года), включающая в себя ряд сгруппированных по целям проектов по формированию системы открытости власти практически во всех сферах деятельности: дорожном хозяйстве, ЖКХ, здравоохранении, инвестиционной деятельности, культуре, образовании, кадровой политики.«Народную карту», которая будет визуализировать в виде отдельных геоинформационных слоев информацию и объекты из всех проектов «дорожной карты».
